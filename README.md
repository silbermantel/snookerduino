**snookerduino**

snookerduino is an arduino based remote controlled scoreboard (specifically designed for snooker referees)

default hardware

* Arduino (pro/micro) compatible 
* 8x8(x4) dotmatrix driven by max7219 
* 7segment array driven by TM 1638
* IR remote 
* IR LED TSOP 4838
